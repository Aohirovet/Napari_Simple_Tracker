# Usage Guide

For installation, see the `Installation` section in [README.md](../README.md). It includes both installation from within napari and installation from a cloned repository with `python -m pip install .`. The plugin assumes that `napari` is already installed in the environment where you use it.

Detailed usage has been split into separate guides for each widget:

- [Simple Tracker Guide](./USAGE_SIMPLE_TRACKER.md)
- [FRAP Analysis Guide](./USAGE_FRAP.md)

These documents describe setup, supported data, outputs, session behavior, and common errors for each workflow.
